# Hayvan Barınağı Takip Sistemi

Bu proje, Kastamonu Üniversitesi Tosya MYO Programlama II dersi dönem sonu ödevi için geliştirilmiş konsol tabanlı bir yönetim sistemidir. 

## Projenin Amacı
Sokak hayvanlarının ve barınaktaki dostlarımızın kayıt altına alınması, sağlık durumlarının takip edilmesi ve sahiplendirilme süreçlerinin dijital ortamda yönetilmesidir.

## Kullanılan Teknolojiler ve Mimari
- **Dil:** Python 3
- **Veritabanı:** JSON (Kalıcı veri depolama)
- **Mimari:** Nesne Tabanlı Programlama (OOP) ve Modüler Tasarım
- **Kavramlar:** Kapsülleme (Encapsulation), Kalıtım (Inheritance), Sınıf ve Nesne (Class/Object)

## Kurulum ve Çalıştırma
1. Proje dosyalarını bilgisayarınıza indirin.
2. Terminal veya komut istemcisinde proje dizinine gidin.
3. Uygulamayı başlatmak için şu komutu çalıştırın:
   ```bash
   python main.py
